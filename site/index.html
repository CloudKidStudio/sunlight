About

Sunlight was developed by Tommy Montgomery. It’s written in JavaScript and provides automatic syntax highlighting to blocks of code contained within an HTML document. It can also serve as plain text to HTML converter. It’s a lot smarter than other syntax highlighters that perform the same task.

To install, simple include the core JavaScript file (sunlight.js) and the desired language files, and then invoke the highlighter with Sunlight.highlightAll();. This will highlight all nodes that have the class sunlight-highlight-{language} where language is the language of the code contained within that node. For example, to highlight PHP code, your raw PHP code would be inside an element with the class sunlight-highlight-php.

See the demos for concrete examples.

For more information on how Sunlight works in addition to the API, consult the documentation.
